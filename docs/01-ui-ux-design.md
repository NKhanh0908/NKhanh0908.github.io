# UI/UX Design Specification

## Design Philosophy

The website should feel:

- Calm
- Professional
- Modern
- Intentional

Animations must support content rather than distract from it.

## Visual Style

Inspired by:

- Linear
- Stripe
- Vercel
- Notion

## Color Palette

Background: #FAFAFA
Surface: #FFFFFF

Primary Text: #111827
Secondary Text: #6B7280

Accent: #2563EB

## Typography

Headings:
- Inter
- 700-800 Weight

Body:
- Inter
- 400-500 Weight

## Design Rules

- Large whitespace
- Strong typography hierarchy
- Minimal borders
- Subtle shadows
- No excessive gradients
- No flashy effects

## Experience Principles

The website should feel:

- Thoughtful
- Technical
- Reliable
- Human

Avoid:

- Hacker aesthetics
- Neon cyberpunk
- Excessive animations
- Dark-only experience
