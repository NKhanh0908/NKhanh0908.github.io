# Animation Specification

## Principle

Subtle First

Animation should never become the main attraction.

## Allowed Effects

Fade

Slide

Scale

Stagger

Parallax

## Forbidden Effects

Heavy rotations

Excessive particles

Flash effects

Aggressive zooming
