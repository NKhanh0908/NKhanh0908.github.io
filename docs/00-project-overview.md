# Project Overview

## Project Name

The Architecture of Ideas

## Vision

A storytelling portfolio website that transforms abstract ideas into visible systems through immersive scrolling experiences.

Instead of presenting information as a traditional resume, the website guides visitors through a visual narrative:

Idea → Structure → Systems → Products → Growth

## Goals

- Showcase technical skills
- Present projects through storytelling
- Create a memorable browsing experience
- Maintain excellent performance and accessibility
- Support future content expansion

## Target Audience

### Primary

- Recruiters
- Hiring Managers
- Technical Leads

### Secondary

- Developers
- Potential Clients
- Students

## Core Principles

- Clean
- Modern
- Bright
- Minimal
- Story-driven
