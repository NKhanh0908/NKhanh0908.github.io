Text:

Hi, I'm Khanh.

Backend-focused Fullstack Developer.

## Scene 02 — Curiosity

The story begins.

Text:

Every developer starts somewhere.

For me, it started with curiosity.

Curiosity about how applications work.

How systems communicate.

How technology solves real-world problems.

Nothing dramatic.
